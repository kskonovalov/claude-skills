# metadata

- **Название:** code-review-types (ревью типобезопасности TypeScript)
- **Автор:** Константин Коновалов
- **Версия:** 1.0.0
- **Описание:** Узкий ревьюер одной линзы — типобезопасности TypeScript. Находит явный и неявный any, небезопасные касты (as Foo без проверки, двойные as unknown as), non-null assertion (!) без гарантии, @ts-ignore/@ts-expect-error, нетипизированные границы (Body без типов, JSON.parse → any), потерю дженерик-параметров и рассинхрон между типом и реальной формой данных в рантайме. Код не редактирует.
- **Применимость:** Используйте при интеграции с внешними API или БД, где легко потерять типизацию; для ревью кода с кастами и any-типами; при настройке строгого tsconfig или переходе на strict-режим.
- **Использование:** Запустить скилл самостоятельно или через оркестратор `code-reviewer`. Анализирует git diff текущей ветки по умолчанию, либо явный путь/diff-файл. Отчёт сохраняется в `<AGENT_HOME>/reports/<YYYY.MM.DD>/code-review-types.md`.
