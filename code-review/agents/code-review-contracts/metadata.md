# metadata

- **Название:** code-review-contracts (ревью API-контрактов)
- **Автор:** Константин Коновалов
- **Версия:** 1.0.0
- **Описание:** Узкий ревьюер одной линзы — согласованности API-контрактов и документации. Сопоставляет DTO ↔ swagger-декораторы ↔ фактическую реализацию, проверяет наличие @ApiProperty/@ApiResponse, ищет рассинхрон фронтенд-типов с бэкенд-контрактом, ломающие изменения существующих эндпоинтов и несогласованный стиль именования полей. Код не редактирует.
- **Применимость:** Используйте при изменении существующих API-эндпоинтов или DTO — чтобы убедиться, что Swagger и фронтенд-типы обновлены синхронно; при добавлении новых эндпоинтов для проверки полноты документации; при рефакторинге полей и ответов на предмет обратной совместимости.
- **Использование:** Запустить скилл самостоятельно или через оркестратор `code-reviewer`. Анализирует git diff текущей ветки по умолчанию, либо явный путь/diff-файл. Отчёт сохраняется в `<AGENT_HOME>/reports/<YYYY.MM.DD>/code-review-contracts.md`.
