# metadata

- **Название:** quick-code-review
- **Автор:** Konstantin Konovalov (оригинал)
- **Версия:** 1.0.0
- **Описание:** Быстрый однопроходный ревью всех ключевых аспектов (баги, безопасность, типы, производительность, читаемость, контракты). Результат — прямо в чат компактной таблицей: файл · метод · проблема · исправление.
- **Применимость:** Когда нужен быстрый взгляд без ожидания 9 параллельных агентов. Не заменяет `code-reviewer` для серьёзного ревью перед мержем — там глубже. Хорошо как первый проход или для небольших фрагментов.
- **Использование:** `/quick-code-review` — ревьюит diff текущей ветки. `/quick-code-review /path/to/file` — конкретный файл или папка. Можно вставить код прямо в запрос.
