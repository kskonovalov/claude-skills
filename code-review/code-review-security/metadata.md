# metadata

- **Название:** code-review-security (ревью безопасности)
- **Автор:** Константин Коновалов
- **Версия:** 1.0.0
- **Описание:** Узкий ревьюер одной линзы — эксплуатируемых уязвимостей по OWASP. Проверяет SQL/NoSQL/command-инъекции, XSS, сломанную аутентификацию/авторизацию, IDOR, хардкод секретов, небезопасную десериализацию, SSRF, небезопасный CORS, утечку PII и подозрительные зависимости. Код не редактирует.
- **Применимость:** Используйте для проверки новых эндпоинтов на уязвимости перед мержем; для ревью кода, работающего с пользовательским вводом, базой данных или внешними запросами; при добавлении новых зависимостей или механизмов аутентификации.
- **Использование:** Запустить скилл самостоятельно или через оркестратор `code-reviewer`. Анализирует git diff текущей ветки по умолчанию, либо явный путь/diff-файл. Отчёт сохраняется в `<AGENT_HOME>/reports/<YYYY.MM.DD>/code-review-security.md` с severity-разбивкой: 🔴 Blocker / 🟠 Major / 🟡 Minor / ⚪ Info.
