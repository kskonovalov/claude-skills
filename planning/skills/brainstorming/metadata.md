# metadata

- **Название:** brainstorming
- **Автор:** Jesse Vincent (@obra) / superpowers, адаптация Константин Коновалов
- **Версия:** 1.0.0
- **Описание:** HARD-GATE перед любым кодом: уточняет требования вопросами по одному, предлагает 2–3 подхода с трейдоффами, получает явное одобрение спека, сохраняет его в `docs/specs/`, передаёт управление в `writing-plans`. Ни строчки кода до одобрения.
- **Применимость:** Обязателен перед созданием новой фичи, компонента, сервиса или любым изменением поведения. Первое звено цепочки `brainstorming → writing-plans → executing-plans`.
- **Использование:** Запускается автоматически при начале реализации чего-либо нового. Задаёт уточняющие вопросы строго по одному, не пачкой. После одобрения спека вызывает `writing-plans`.
