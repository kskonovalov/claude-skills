# metadata

- **Название:** executing-plans
- **Автор:** Jesse Vincent (@obra) / superpowers, адаптация Константин Коновалов
- **Версия:** 1.0.0
- **Описание:** Исполняет план из `docs/plans/` шаг за шагом: каждая задача — отдельный субагент (в Claude Code) или инлайн (в других средах). После каждой задачи — чекпоинт. Останавливается при блокере, не перепрыгивает шаги, не пропускает проверки.
- **Применимость:** Используйте когда план написан и одобрен. Третье звено цепочки `brainstorming → writing-plans → executing-plans`. Также можно запустить напрямую если план уже готов.
- **Использование:** Загружает план, критически проверяет его, затем исполняет задачи. В Claude Code — через `Agent` tool с `run_in_background: false` для последовательности. При блокере — стоп и доклад, не попытка обойти.
