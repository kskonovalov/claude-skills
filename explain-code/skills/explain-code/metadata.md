# metadata

- **Название:** explain-code
- **Автор:** Константин Коновалов
- **Версия:** 1.0.0
- **Описание:** Анализирует конкретный код — функцию, модуль, компонент, фичу — и выдаёт структурированный отчёт: назначение, пошаговое объяснение работы, поток данных, ключевые зависимости, неочевидные места. Привязывает каждое утверждение к файлу и строке.
- **Применимость:** Используйте когда нужно быстро разобраться в незнакомом или давно не тронутом коде: «как работает X», «объясни этот хук/сервис/компонент», «покажи data flow», «walk me through». Не для поиска багов (→ `systematic-debugging`) и не для ревью качества (→ `code-reviewer`).
- **Использование:** Задать вопрос про конкретный код — скилл сам найдёт файл/символ через grep, прочитает нужные части и сформирует отчёт. Язык отчёта — язык запроса, дефолт русский.
